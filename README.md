# PR早見帳 — Premiere Pro ショートカット & 便利ワザ

Premiere Pro のショートカットをスマホで検索できる早見帳PWA。
動画編集CAMP講座の推奨キーマップをベースに、実際に使用中のキーボードショートカット設定（.kys）から生成。

- アプリ: `index.html`（自己完結・ビルド不要）
- 使い方: `manual/index.html`
- ホーム画面に追加するとオフラインでも使えます（PWA）

Inspired by [AE早見帳](https://tkwfunkypop.github.io/public/tools/ae-shortcuts/).
